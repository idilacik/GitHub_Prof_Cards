# Github Profil Kartları
"Github Profil Kartları", kullanıcıların isim, soyisim veya kullanıcı adı </br> ile Github profillerini arayabilmesini sağlayan bir web uygulamasıdır.</br> Aranan profilin takipçi, takip edilen ve repo sayısını gösterir ve üç adet </br> repoyu isimleri ve doğrudan bağlantılarıyla birlikte sunar.

## Özellikler
### Arama Fonksiyonu: </br> 
İsim, soyisim veya kullanıcı adı kullanarak Github profillerini arama.
### Profil Bilgileri: </br>
Aranan profilin takipçi sayısını, takip edilen kişi sayısını ve repo sayısını gösterir.
### Repo Gösterimi: </br>
Kullanıcının profilinden üç repoyu isimleri ve bağlantılarıyla birlikte gösterir.
